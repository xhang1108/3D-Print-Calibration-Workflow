Input Shaping

What is Input Shaping?

Input Shaping is a technique used to reduce ringing and ghosting in 3D prints by compensating for printer vibrations.

Calibration Process

  1. Enable input shaping in your firmware
  2. Print a ringing test pattern
  3. Measure the ringing frequency
  4. Calculate shaper parameters
  5. Apply the parameters in your firmware
  6. Print another test pattern to verify improvements

Benefits of Input Shaping

  • Reduced ringing and ghosting
  • Ability to print at higher speeds
  • Improved overall print quality
  • Less need for physical printer modifications
