Skew Correction

What is Skew?

Skew occurs when your printer's axes are not perfectly perpendicular to each other, resulting in prints that are not square or rectangular. Correcting skew improves dimensional accuracy.

Correction Process

  1. Print a calibration cube or skew correction model
  2. Measure the dimensions and diagonals of the printed object
  3. Calculate skew factors using the measurements
  4. Apply skew correction in firmware or slicer settings
  5. Print another test object to verify the correction

Causes of Skew

  • Misaligned frame
  • Loose belts or pulleys
  • Bent lead screws
  • Improperly squared gantry

Tips

  • Address mechanical issues before applying software corrections
  • Use precision measuring tools for accurate results
  • Re-check skew periodically, especially after moving the printer
  • Consider using a 3D touch probe for automated skew correction (if supported by your firmware)
